이 메모장에 대하여

이 웹 메모장은 온라인에서 간단하게 메모를 작성하고 저장할 수 있는 도구입니다. 별도의 로그인 없이도 자동 저장 기능을 제공하며, 다크 모드를 통해 눈의 피로를 줄일 수 있습니다.

주요 기능

  • 입력 내용 자동 저장 (브라우저 로컬 저장소 사용)
  • 복사, 텍스트 파일 저장 기능 제공
  • 다크 모드 지원
  • 실시간 언어 감지 (한글 또는 영어)

사용 팁

중요한 메모를 작성한 뒤 "Save as .txt" 버튼을 눌러 텍스트 파일로 백업해 두세요. 장시간 메모 시에는 다크 모드를 켜면 눈의 피로를 줄일 수 있습니다.
